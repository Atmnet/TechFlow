## ✨ 新功能

### 1. YAML 导入功能
**功能描述**: 支持粘贴 K8s YAML 配置，自动解析为可视化资源

**实现细节**:
- ✅ 使用 `js-yaml` 库解析 YAML
- ✅ 支持多文档 YAML（`---` 分隔）
- ✅ 自动识别资源类型：
  - Pod
  - Service
  - Deployment
  - ConfigMap
  - Secret
  - Database
- ✅ 自动提取配置信息：
  - 镜像 (image)
  - 端口 (port)
  - 副本数 (replicas)
- ✅ 智能布局：自动排列节点位置
- ✅ 自动连接：根据 selector 自动建立连接关系

**使用步骤**:
```
1. 点击左侧"📥 导入 YAML"按钮
2. 粘贴 K8s YAML 配置
3. 点击"导入"按钮
4. 自动生成可视化资源和连接
```

**示例 YAML**:
```yaml
apiVersion: v1
kind: Pod
metadata:
  name: web-app
spec:
  containers:
  - name: nginx
    image: nginx:1.21
    ports:
    - containerPort: 80
---
apiVersion: v1
kind: Service
metadata:
  name: web-service
spec:
  selector:
    app: web
  ports:
  - port: 80
    targetPort: 80
```
