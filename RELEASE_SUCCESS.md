## 🎯 v0.2.3 版本亮点

### ✨ 新功能

1. **K8s 可视化编辑器**
   - 拖拽式资源管理
   - 支持 6 种资源类型
   - 实时 YAML 生成
   - YAML 导入/导出

2. **Dual-View 播放器**
   - 视频 + 代码同步
   - 时间轴联动
   - Monaco Editor 高亮

3. **YAML 验证服务**
   - 语法验证
   - 结构验证
   - 批量验证

### 🔧 技术栈

- Next.js 14.2.3
- React 18.3.1
- TypeScript 5.4.5
- Fastify 4.26.2
- Framer Motion 11.11.1
- Monaco Editor 4.6.0

## 🚀 快速开始

### 克隆仓库

```bash
git clone https://github.com/Atmnet/TechFlow.git
cd TechFlow
```

### 启动前端

```bash
cd frontend
npm install
npm run dev
# 访问 http://localhost:3000
```

### 启动后端

```bash
cd backend-poc
npm install
npm run dev
# 访问 http://127.0.0.1:3001
```
